NDYL | Data Technician Apprenticeship Support (ST0795)

Welcome to NDYL's support hub for the Level 3 Data Technician (ST0795) apprenticeship. This resource is designed to guide apprentices through Assessment Method 1: Scenario Demonstrations with Questioning, using practical, skills-focused examples.

Based on the official IfATE Data Technician Standard , each section supports real skills development using tools like Excel, Power BI, and Google Sheets.
